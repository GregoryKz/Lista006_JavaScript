Lista 06 de Exercícios - Estrutura de Dados Vetor
Descrição

Esta lista contém 15 exercícios de lógica de programação focados no uso de vetores (arrays) em JavaScript. Os exercícios envolvem:

Leitura e exibição de elementos

Inversão, ordenação crescente e decrescente

Soma, média e contagem de valores (pares, positivos, negativos)

Verificação de condições (presença de zero, intervalos específicos)

Manipulação de nomes e exibição em ordem alfabética

O objetivo é praticar o uso de vetores, desenvolvendo raciocínio lógico, armazenamento e manipulação de dados e aplicação de operações sobre coleções de elementos.
